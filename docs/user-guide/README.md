# User Guide

This guide covers practical usage of the Cyberstar Simulator — how to create a show, edit it, export it, and get it into SPTE.

---

## Quick Orientation

When you open `index.html` in a browser you see the piano roll editor:

```
+------------------------------------------------------------------+
|  TOOLBAR                                                         |
|  [New Show] [Save] [Saved Shows] [Load WAV]                      |
|  [Play] [Pause] [Stop]  [Snap] [State] [Zoom]                    |
|  [4ch Tester]  [Export 4ch WAV]                                  |
+------------------------------------------------------------------+
|  CHARACTER   |  TIMELINE (scrollable, zoomable)                  |
|  LABELS      |  +--+--+--+--+--+--+--+--+--+--+----->           |
|              |  |  signal blocks (drag to create)                |
|  Rolfe       |  |  ####     ###        ##                        |
|  Earl        |  |       ###                  ####                |
|  Dook LaRue  |  |  ...                                           |
|  Fatz        |  |                                                |
|  Beach Bear  |  |                                                |
|  Looney Bird |  |                                                |
|  Mitzi       |  |                                                |
|  Billy Bob   |  |                                                |
|  Lights      |  |                                                |
|              |  |                                                |
+--------------+--------------------------------------------------+
```

The **4ch Tester** button opens a modal overlay with a live LED stage preview and WAV validation tools.
